Project B

Project B is my thesis project, where I developed a multiplayer game using Monogame and SignalR. The main goal was to create something using backend, frontend, database, and testing. It's a 2D game where you are able to run around and see other players in real time. Although the game is simple, the goal was to demonstrate the use of SignalR in a game. The server is hosted on Heroku, and the database is MongoDB. I used Docker for deployment and GitHub Actions for CI/CD. Although the server is currently down, you can still download the client and play it offline locally without issues.

Project X

A group project where we created a cmd/terminal RPG game in C# using.Net, MariaDB, and Dapper. The main goal was to create something challenging as we were limited by the terminal. You play as a hero who battles in the arena and earns gold based on his performance. You can buy weapons and armor to help you in your journey. The game is played in the terminal and is controlled by keyboard inputs. I later refactored the game to use Docker with Evolve to start the MariaDB server. NOTE: If you wish to play the game, you require Docker to run the MariaDB server.
